====== void ReproduceAgent(int t,int sourcei,int sourcej,int desti,int destj); ======
Created Friday 31 January 2014

A new agent is created from a parent at position sourcei, sourcej, with the function [[Agent.hh:void CreateAgentFromParent(Agent  A int agid int t)|CreateAgentFromParent(P,pagidc_,t).]] This new agent is placed in the Ags field at position desti, destj. Remember: this agent will immediately be developed as well.
